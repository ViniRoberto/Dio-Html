Entendendo sobre a tag P:

Voltar

A tag HTML P é utilizada para definir um parágrafo, ou seja, um bloco de texto. É um elemento de fluxo que pode ser utilizado em qualquer lugar que se espere um elemento de fluxo

Como funciona as tags H1,H2,H3...

Voltar

As tags H1, H2 e H3 são tags de programação HTML que estruturam e hierarquizam o conteúdo de uma página. Elas são usadas para destacar títulos e subtítulos, facilitando a leitura e a compreensão do conteúdo.
